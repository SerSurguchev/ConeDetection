# Аннотация
Целью этого проекта является подготовка данных для обучения и само обучение архитектур нейронных сетей (YOLOv4 darknet, YOLOv5, Faster-RCNN Detectron2) на предмет распознавания конусов на гоночной трассе.  

Сам датасет и координаты Bounding Box к нему доступны в репозитории MIT: https://github.com/cv-core/MIT-Driverless-CV-TrainingInfra/tree/master/CVC-YOLOv3
